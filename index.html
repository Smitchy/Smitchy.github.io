Lego World VR

Lego World VR is my bachelor project in association with Lego. The project is developed in Unity and coded in C#.
This project aims to provide an experience based within a Lego world, where the user is able to explore Lego cities online with friends as well as participate in minigames. Key aspects of the project include:

  • Vast Lego cities
  • Networked multiplayer
  • Voice chat
  • Online hosted maps

During this project i have been working on the networking aspects of the game. The multiplayer element was created using the Photon SDK. I also implemented online streaming of the scenes using Addressable assets and an AWS bucket.

Litter Bay

This project was created as part of Nordic Game Jam 2019.
In Litter Bay two players face each other in 2 rounds.
Each round the players race to the center of the map.
First round players control a trash cleaning drone that picks up trash where it moves.
Second round the player swap position and control a jetski that gets blocked by trash in the water and therefore have to follow the path the opponent created round one.
The game is made in Unity
